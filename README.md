CreditWise - Loan Approval Prediction System

## Project Overview
CreditWise is a Machine Learning project that predicts whether a loan application should be approved or rejected. The system uses historical loan application data to train a Logistic Regression model for accurate predictions. It helps banks and financial institutions make faster and more consistent decisions while reducing human error.

## Features
- Predict loan approval (Approved / Rejected)
- Train and test using historical data
- Evaluate model using accuracy and confusion matrix
- Predict new applicants instantly

## Dataset
The dataset includes features such as:
- Gender
- Married
- Education
- Self_Employed
- ApplicantIncome
- LoanAmount
- Credit_History
- Loan_Status (Target variable: Approved / Rejected)

## Technology Stack
- Python 3.14
- Libraries: pandas, numpy, matplotlib, seaborn and scikit-learn
- Environment: VS Code / Jupyter Notebook
